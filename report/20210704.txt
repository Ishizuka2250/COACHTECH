説明リスト
→ <dl>タグ内に用語を表す<dt>と用語の説明を表す<dd>を使って説明リストを作成する

リンクタグ
→ <a href="リンク先"></a>
→ target="_blank"をつけることで新しいタブで開く

画像タグと動画タグ
画像タグ
→ <img src="画像ファイルパス" alt="代替テキスト">
サイズを変える場合
→ <img src="画像ファイルパス" width="幅" height="高さ">
動画タグ
→ <video src="動画ファイルパス" controls></video>
※動画再生パネルを表示する場合はcontrols属性が必要
※自動再生する場合は autoplay属性を追加する (chromeの場合はmuted属性も必須)
※動画をループさせる場合は loop属性 を追加する
→ width属性とheight属性で動画ファイルのサイズを変更できる

テーブルタグ
→ <th>でテーブルの見出しを定義できる
→ colspan属性で水平方向に結合
→ rowspan属性で垂直方向に結合

フォームタグ
<form action="送信先" method="get(初期値) / post">
</form>
get:サイト内検索
post:送信用入力フォーム
<input type="部品属性値" name="部品名" value="初期値 / 送信データ / ボタンのラベル"
type一覧:
text - テキストフィールド
tel - 電話番号フィールド
url - URLフィールド
email - メールアドレスフィールド
search - 検索フィールド
date - 日付入力
password - パスワード入力フィールド
radio - ラジオボタンフィールド
checkbox - チェックボックスフィールド
file - ファイル送信用ボタン
submit - 送信ボタン
複数行テキスト入力フォーム
<textarea name="部品名" cols="1行に表示するおおよその行数" cols="表示行数"></textarea>
optionタグはvalueで送信する値を決定する
<option value="送信する値">プルダウンに表示する値</option>

label：
ラベルをクリックすることで関連付けた部品も選択可能となる
視覚的にWebサイトを閲覧できない人にもどのラベルと部品が関連付けられているか明確化することができる

グループ化タグ
div span
→ タグ自体に意味はない cssの修飾で使用する
div：ブロック要素, widthとheightの指定可能, 前後に改行が入る
span：インライン要素, widthとheightの指定不可, 前後に改行が入らない

セクション
<section>
章・節・項などのセクションを表す
<article>
ブログやニュース記事など、単体でコンテンツとして成立するものを囲む
<header>
ページの先頭に表示されるロゴやナビゲーションなどに利用する
<main>
そのページの主要コンテンツに利用する (ロゴやナビゲーションバーなどに囲ってはいけない)
<footer>
そのページの末尾のコンテンツ(連絡先、著作権、関連リンク)に使用する
<aside>
補足的なセクションを表すタグ(サイドバーとかに使用)
<nav>
ナビゲーションを表すタグ (全ページに一貫して表示されるグローバルナビゲーションに使用)
